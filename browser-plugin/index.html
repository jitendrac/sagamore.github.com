Sagamore Plugin

The Sagamore Plugin allows the Sagamore POS to print to your receipt printer and access other POS devices.

Setup Instructions

  1. Download the Sagamore Browser Plugin installer (below). If prompted, click "Save File" or "Save".
  2. Double click the file (SagamoreBrowserPlugin_Installer.msi) in your Downloads window to run it.
  3. Follow the instructions to complete installation.
  4. IMPORTANT: Close and reopen any web browsers (Firefox, Google Chrome, Internet Explorer) before attempting to print.
